# БСПб Кейс-турнир

Проект выполнен в рамках кейс-турнира **БСПБ (Банк Санкт-Петербург)**.  
Работа объединяет SQL и Python части анализа, направленные на исследование активности пользователей интернет-банка и используемых платформ.

---

## О проекте

В рамках кейса необходимо было:
- изучить данные о входах пользователей в интернет-банк и их активности
- рассчитать MAU и отследить его динамику
- найти аномалии и объяснить возможные причины

---

## Стек технологий

SQL, Python, Pandas, Matplotlib, Seaborn, NumPy

---

## Ключевые выводы

- Android-X - основная платформа, формирующая большую часть пользовательской активности.  
- Более 95% пользователей демонстрируют низкую активность (менее 5 входов в месяц).  
- Переход между версиями повлиял на временное снижение MAU в январе.
